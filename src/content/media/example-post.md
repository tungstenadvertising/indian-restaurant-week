This is an example media post. You can create new posts by adding markdown files to the `src/content/media/` directory.

## Features

- Write posts in Markdown
- Add images, links, and formatted text
- Tag your posts for easy organization
- Automatic date sorting

## Getting Started

To create a new post:

1. Create a new `.md` file in `src/content/media/`
2. Add frontmatter with title, date, excerpt, etc.
3. Write your content in Markdown
4. The post will automatically appear on the media page!
